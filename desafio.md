# Mini Desafio — Conversão de Temperatura (Função Tradicional)

Este desafio consiste em criar uma função tradicional em JavaScript para converter temperaturas de Celsius para Fahrenheit.

---

## Objetivos do Desafio
- Praticar funções tradicionais.
- Demonstrar retorno de valores.
- Utilizar escopo local.
- Validar entradas recebidas.
- Executar testes no console.

---

# O Desafio

Crie uma função tradicional chamada:

### converterCelsiusParaFahrenheit(celsius)

Fórmula:
F = (C × 9/5) + 32

A função deve:
1. Receber um valor em Celsius.
2. Verificar se o valor é numérico.
3. Calcular o valor correspondente em Fahrenheit.
4. Retornar o resultado convertido.

---

## Tarefas
1. Implementar a função usando somente a sintaxe tradicional `function`.
2. Validar a entrada (ex.: impedir strings, null, undefined).
3. Retornar o valor convertido.
4. Criar testes no console, como:

```js
converterCelsiusParaFahrenheit(0)
converterCelsiusParaFahrenheit(25)
converterCelsiusParaFahrenheit(100)
