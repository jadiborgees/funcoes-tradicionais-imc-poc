# Funções Tradicionais — PoC (Calculadora de IMC)

**Tema:** Funções Tradicionais  
**Aluno:** Jadi Borges

## Objetivos
- Entender funções tradicionais em JavaScript.
- Mostrar escopo local e global.
- Demonstrar retorno de valores.
- PoC: Implementar uma calculadora de IMC (Índice de Massa Corporal).

## Estrutura do repositório
- `index.html` — página simples para executar os exemplos.
- `script.js` — código comentado com exemplos básicos e avançados.
- `desafio.md` — mini-desafio proposto para colegas.

## Exemplos incluídos
1. Função básica (sem retorno) — mostra escopo local.
2. Função com retorno — cálculo do IMC.
3. Exemplo avançado — função que valida entrada e retorna objeto com resultado.
4. Demonstração de escopo global vs local.

## Como executar
1. Abra `index.html` no navegador (duplo clique ou via GitHub Pages).
2. Use a área interativa para testar valores de peso e altura.
3. Veja `script.js` para comentários detalhados.

## Mini-desafio
Veja `desafio.md` para enunciado e critérios de correção.
